# tf-faster-rcnn

```yaml

.
├── data     # 存放数据集

│

│
├── models              # 存放网络模型
│    └── ..
│
├── config.py           # 设置
├── data_loader.py      # 处理数据的读入
├── train.py            # 训练模型

├── README.md           
└── LICENSE             

```
